## 📋 Sobre o projeto

Este projeto faz parte da minha jornada de aprendizado no curso de Tailwind CSS do professor Matheus Fraga na Udemy. O Clone Uber é uma recriação fiel da landing page da Uber, explorando na prática os recursos do Tailwind CSS para criar interfaces modernas, responsivas e adaptadas para múltiplos tamanhos de tela.

---

## ✨ Funcionalidades

- 📱 **Responsividade completa** — adaptado para mobile, tablet e desktop
- 🎨 **Fiel ao design original** — layout e identidade visual da Uber
- 🧭 **Navbar** — com links de navegação e botões de Login e Sign up
- 🖼️ **Hero section** — imagem de fundo em tela cheia com card de ações sobrepostos
- 🃏 **Cards de ações** — Faça viagens ou entregas, Viajar e Alugue seus carros

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura e semântica da página |
| Tailwind CSS | Estilização e responsividade com classes utilitárias |

---

## 📚 Conceitos praticados

- Classes utilitárias do Tailwind CSS
- Responsividade com breakpoints do Tailwind (`sm`, `md`, `lg`, `xl`)
- Layouts com Flexbox via Tailwind
- Posicionamento e sobreposição de elementos
- Deploy na Vercel
